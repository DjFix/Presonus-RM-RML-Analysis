IOFWCmdQ

Structure for head of a queue of IOFWCommands 

struct IOFWCmdQ { 
    IOFWCommand *fHead; 
    IOFWCommand *fTail; 
    bool executeQueue(
        bool all); 
    virtual void headChanged(
        IOFWCommand *oldHead); 
};
Field Descriptions
fHead
Points to the head of the queue, or NULL if queue is empty
fTail
Points to the tail of the queue, or NULL if queue is empty
