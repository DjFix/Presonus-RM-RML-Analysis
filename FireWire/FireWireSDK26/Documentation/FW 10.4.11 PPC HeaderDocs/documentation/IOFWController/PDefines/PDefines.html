Discussion

gIOEthernetWakeOnLANFilterGroup is an OSSymbol object that contains the name of the FireWire wake-on-LAN filter group defined by kIOFWWakeOnLANFilterGroup.

kIOFWAddressSize

The number of bytes in an FireWire hardware address. 

#define kIOFWAddressSize 8

kIOFWControllerClass

kIOFWControllerClass is the name of the IOFWController class. 

#define kIOFWControllerClass "IOFWController"

kIOFWCRCSize

The size in bytes of the 32-bit CRC value appended to the end of each FireWire frame. 

#define kIOFWCRCSize 0

kIOFWMaxPacketSize

The maximum size of an FireWire packet, including the FCS bytes. 

#define kIOFWMaxPacketSize 4096

kIOFWMinPacketSize

The minimum size of an FireWire packet, including the FCS bytes. 

#define kIOFWMinPacketSize 64

kIOFWWakeOnLANFilterGroup

kIOFWWakeOnLANFilterGroup describes the name assigned to the FireWire wake-On-LAN filter group. 

#define kIOFWWakeOnLANFilterGroup "IOFWWakeOnLANFilterGroup"
