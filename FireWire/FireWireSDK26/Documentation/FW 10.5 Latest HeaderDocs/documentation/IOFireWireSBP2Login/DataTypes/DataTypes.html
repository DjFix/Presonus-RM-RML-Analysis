FWSBP2StatusBlock

typedef struct { 
    UInt8 details; 
    UInt8 sbpStatus; 
    UInt16 orbOffsetHi; 
    UInt32 orbOffsetLo; 
    UInt32 status[6]; 
} FWSBP2StatusBlock;
Fields
details
Src, Resp, D, Len fields of status block format
sbpStatus
SBP2 specific status
orbOffsetHi
High 32 bits of address of orb status is for.
orbOffsetLo
Low 32 bits of address of orb status is for.
status
[6] Up to 48 bytes of additional data. Length is determined by len field.
