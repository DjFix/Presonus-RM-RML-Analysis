Enumerations


Enumeration of wake-On-LAN filters.

enum { 
    kIOFWWakeOnMagicPacket = 0x00000001, 
    kIOFWWakeOnPacketAddressMatch = 0x00000002 
};
Constants
kIOFWWakeOnMagicPacket
Reception of a Magic Packet.
kIOFWWakeOnPacketAddressMatch
Reception of a packet which passes through any of the address filtering mechanisms based on its destination FireWire address. This may include unicast, broadcast, or multicast addresses depending on the current state and setting of the corresponding packet filters.
Discussion

An enumeration of all filters in the wake-on-LAN filter group. Each filter listed will respond to a network event that will trigger a system wake-up.
